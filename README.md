# 🧮 Calculator App (useReducer)

A calculator application built with React using useReducer for state management.

## 🚀 Features
- Basic math operations (+, -, *, /)
- State management with useReducer
- Component-based architecture
- Clean and responsive UI

## 🛠️ Technologies
- React
- JavaScript
- CSS

## ⚙️ Installation

```bash
npm install
npm run dev
```
